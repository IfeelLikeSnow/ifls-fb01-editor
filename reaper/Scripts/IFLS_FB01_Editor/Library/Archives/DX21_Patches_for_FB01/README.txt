These are the factory standard patches from the DX-21 converted for use on the FB-01.  I converted them using Martin Tarenskeen's FB_CNVRT program for the AtariST.  So a big thanks to him for that very useful program!  I also included a couple of the cooler DX-100 voices that weren't on the DX-21.  Otherwise all the other DX-100 voices are either the same as the DX-21 or rather mediocre.

Files included in this archive:
-------------------------------

DX21-1B1.SYX
DX21-2B2.SYX
DX21-3B2.SYX

The files ending in "B1" load to the first bank of the FB-01.  The files ending in "B2" load to the second bank of the FB-01.
