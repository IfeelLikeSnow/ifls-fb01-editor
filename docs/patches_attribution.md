# Patch Libraries Attribution

This project can optionally use third‑party Yamaha FB‑01 SysEx libraries found online.
Licensing/redistribution terms vary by source and are often not explicitly stated.

## How to document a library
For each library you include locally, add:
- Library name
- Original source URL
- Author/collector (if known)
- License/permission statement (if present), otherwise: "Not stated"
- Date accessed

## Takedown
If you are a rights holder and believe a library should not be redistributed, please open an issue or contact the maintainer and it will be removed from release assets.


## Note on 'freely downloadable'
Even if libraries are widely mirrored online, redistribution rights may not be explicitly stated. This document helps track sources and permissions.
